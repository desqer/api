    create(conn, map)

    Creates user.

    Endpoint example:

    POST /users

    Params example:

    {
  "user": {
    "name": "John Doe",
    "phone": "5547999874321",
    "password": "foo123"
  }
}

    Success response code 200 example:

    {
  "data": {
    "id": "2050ea22-a273-4bef-93e9-ce9df0e73ddc",
    "name": "John Doe",
    "phone": "55479995874321",
    "email": null,
    "bio": null,
    "professional": false,
    "confirmed": false,
    "deleted": false
  }
}

    Error response code 422 example:

    {
  "errors": {
    "phone": ["has already been taken"],
    "password": ["can't be blank"]
  }
}
